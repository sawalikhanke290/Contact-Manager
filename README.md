# Contact-Manager
A Contact Manager application built with React.js for managing contacts, featuring add, edit, delete functionalities, and a simulated backend using JSON Server for data persistence.

 Features

- Add, edit, and delete contacts
- Search and filter contacts
- Responsive design
- Simulated backend with JSON Server

  The application should now be running at http://localhost:3000 and the JSON Server at http://localhost:7000

Technologies Used
Frontend: React.js, Bootstrap
Backend: JSON Server
